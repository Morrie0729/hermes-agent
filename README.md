# Hermes Agent

An intelligent AI assistant created by **Nous Research**.

## About

Hermes is a helpful, knowledgeable, and direct AI assistant that can:

- Answer questions and provide insights
- Write and debug code
- Execute terminal commands
- Search files and web content
- Analyze data and images
- Manage tasks and workflows
- Connect to external services (GitHub, databases, APIs)

## Capabilities

- **Persistent Memory**: Remembers user preferences and context across sessions
- **Multi-Platform**: Works via CLI, Telegram, Discord, and more
- **Skill System**: Learns and reuses successful workflows
- **Vision**: Can analyze images and screenshots
- **Cron Jobs**: Schedule automated tasks
